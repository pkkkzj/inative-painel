# Inative Painel

Version: 2.7

Inative Painel is a Roblox utility panel focused on fast access to gameplay tools, visual options, avatar modifiers, and session utilities through a clean in-game interface.

## Features

- Modern draggable UI with minimize/restore support
- Settings page with theme customization
- Dark and white theme modes
- Custom accent color
- Config saving for preferences
- Movement tools
  - Fly
  - VFly
  - Noclip
  - Speed Boost
  - Jump Boost
  - Walk Mode keybind
- Combat tools
  - God Mode
  - Aimbot
  - Bot Aimbot
  - MM2 Aimbot + Hitbox
  - Phantom Mode
  - Fling
  - Auto Parry
- Visual tools
  - ESP Highlight
  - ESP Name
  - ESP Box
  - ESP Lines
  - ESP Distance
  - ESP Health
  - Role / Item Tags
  - Sheriff / Murderer ESP
  - Dropped Sheriff Gun ESP
  - Computer ESP
  - Generator ESP
- World tools
  - Full Bright
  - Map Shader
  - Unlimited Zoom
  - No Fall Damage
  - Anti Fling
  - Infinite Jump
- Freecam and session tools
  - Free Cam
  - Click Teleport
  - Lock Emote
  - Safe Position save/return
  - Server Rejoin
  - Server Hop
- Avatar tools
  - Korblox Leg
  - Headless
  - Animation packs
  - Individual animation selector
  - Restore original animations
- Panel member tag system
  - Toggle in Settings to enable or remove head tags
